Crypto Investment Manager

Overview
Crypto Investment Manager is a data-driven application that helps users analyze cryptocurrency markets and make informed investment decisions. The system performs risk analysis, price prediction using machine learning, and suggests an optimal investment portfolio.

---

Features

- 📊 Real-time cryptocurrency data fetching
- 📈 30-day price trend analysis
- ⚠️ Risk calculation (Volatility)
- 📉 Expected return & Sharpe ratio
- 🤖 Price prediction using Linear Regression
- 💼 Portfolio allocation based on risk levels
- 📊 Data visualization (graphs & charts)
- 📄 Report generation (`crypto_report.txt`)
- 🌐 Interactive dashboard using Streamlit
